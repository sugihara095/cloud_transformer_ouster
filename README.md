# cloud_transformer_ouster

このパッケージは、AutowareのGround Filter等を適用しやすくするために、点群データの座標をセンサーの高さ分Z軸方向にオフセットし、地面の高さをZ=0に合わせるためのROS 2ノードです。

本パッケージはバックパック型LiDARの統合処理パイプラインの一部として動作します。
環境構築や具体的な使用方法については、以下のメインリポジトリを参照してください。

**[backpack-LiDAR-intensity-correction のリポジトリへ](https://github.com/sugihara095/backpack-LiDAR-intensity-correction)**
